## collection.Remove  
  
  
**Description:** Removes entities from a collection.

  
  
**Parameters:**  
  * *coll:* The collection to be updated.  
  * *entities:* Points, polylines, polygons, and collections to add. Or null to empty the collection.  
  
**Returns:** void
