## poly2d.Union  
  
  
**Description:** Create the union of a set of polygons. The original polygons are not edited.  
  
**Parameters:**  
  * *entities:* A list of polygons, or entities from which polygons can be extracted.  
  
**Returns:** A list of new polygons.
