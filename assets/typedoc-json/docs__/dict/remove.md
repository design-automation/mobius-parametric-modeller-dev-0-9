## dict.Remove  
  
  
**Description:** Removes keys from a dict. If the key does not exist, no action is taken and no error is thrown.

  
  
**Parameters:**  
  * *dict:* The dict in which to remove keys  
  * *keys:* The key or list of keys to remove.  
  
**Returns:** void
